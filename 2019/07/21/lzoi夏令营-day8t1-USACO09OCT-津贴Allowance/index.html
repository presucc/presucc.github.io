【题目描述】

作为创造产奶纪录的回报，Farmer John决定开始每个星期给Bessie一点零花钱。

Farmer John有一些硬币，一共有$N$ $(1 <= N <= 20)$种不同的面额。每一个面额都能整除所有比它大的面额。

他想用给定的硬币的集合，每个星期至少给Bessie某个零花钱的数目$C$ $(1 <= C <= 100000000)$。请帮他计算他最多能支付多少个星期的零花钱。

【输入格式】

第$1$行: 两个由空格隔开的整数: $N$ 和 $C$；

第$2$到第$N+1$行: 每一行有两个整数表示一个面额的硬币：硬币面额$V$ $(1 <= V <= 100,000,000)$拥有的该面额的硬币数$B$ $(1 <= B <= 1,000,000)$。

【输出格式】

一个单独的整数，表示最多能给支付多少个星期至少为$C$的零用钱。

【提示】

每个星期至少给Bessie的零花钱为$6$。有$100$个面额为$1$的硬币，$120$个面额为$5$的硬币，和$1$个面额为$10$的硬币。可以在第一个星期超额付给Bessie$1$个$10$元的硬币，然后接下来的$10$个星期每星期付给Bessie$2$个$5$元的硬币。最后$100$个星期每星期付给Bessie$1$个$1$元的硬币和一个$5$元的硬币。
