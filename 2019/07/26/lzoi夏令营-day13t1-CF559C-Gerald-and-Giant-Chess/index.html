题目描述

$C$酱制作了一个$n*m$的棋盘。棋盘上有一些格子被涂成了黑色。他想知道，若只能向下走或者向右走，有多少种方案能从左上角走到右下角（即从$(1,1)$走到$(n,m)$），且不经过黑色的格子。

输入格式

第一行为三个整数$n,m,k$，其中k表示黑色格子的个数。

接下来k行，每行为两个整数$x,y$，表示$(x,y)$是黑色格子。

输出格式

输出一行一个整数，表示方案数对$10^9+7$取模的结果。

样例输入2

样例输出1

样例输出2

提示

对于$20$%的数据，保证$1≤n,m≤10$。

对于$40$%的数据，保证$1≤n,m≤2000$。

对于另外$30$%的数据，保证$k=0$。

对于$100$%的数据，保证$1≤n,m≤10^5$，$0≤k≤2000$，$1≤x≤n$，$1≤y≤m$。保证$(1,1)$和$(n,m)$是白色格子，给出的$(x,y)$不会重复。

做法

组合数学 + 动态规划

因为$1≤n,m≤10$，数据范围很小，所以我们可以尝试着爆搜。

$100$分做法：

$40$分的$O(nm)$用不了了，但是我们可以以此为基础，继续寻找优化的方式。

正难则反，考虑用总方案数减去至少经过一个黑格子的路线数量即为答案。

重点是如何做到不重不漏。把所有格子按照横纵坐标排序。假设右下角是第 n + 1 个格子，

设 $f[i]$ 表示从左上角走到排序后的第 $i$ 个点，并且中途不经过其他黑格子的路线数。则有：

$$ f[i] = foto(1, 1, a[i].x, a[i].y) - \sum_{j = 1}^{i - 1} f[j] * foto(a[i].x, a[i].y, a[j].x, a[j].y) $$

这句话的意思是：枚举的 $j$ 作为从左上角到第 $i$ 个点经过的第一个黑格子，总路线数为 $f[i]$ ，然后到第 $i$ 个点就可以任意走，再根据加法原理以及乘法原理可得所有非法方案，并且可以发现第一个经过得黑格子不同，则不同的 $j$ 对应得路线一定不同，而 $j$ 又取遍了所有 $i$ 之前得黑格子，所有没有遗漏。

首先，我们枚举到第一个红点$(2, 2)$， 计算$foto(1, 1, 2, 2) = 2$，$f[1]$先$ += 2$，即所有从起点到该点的路线数为$2$；

我们发现这个点$i = 1$连第二层循环都进不了，继续枚举下一个$i$。

为了方便，我们在第一个红点那里标上一个$2$。根据上面的枚举，$f[1] = 2$，即有$2$条以该点为第一个经过的不合法点的路径。

第二个红点$(2, 3)$，计算$foto(1, 1, 2, 3) = 3$.,$f[2]$先$ += 3$，即所有从起点到该点的路线数为$3$；

进入第二层循环。
$j = 1$时，$f[2] -= f[1] * foto(2, 2, 2, 3)$，即减去了一些先经过了第一个不合法点再经过该点的路径。对这个点来说，这些被减去的路径是不合法的。

此时, $f[2] = 3 - 2 = 1$；

我们在第一个红点那里标上一个$2$。根据上面的枚举，$f[1] = 2$，即为有$2$条以该点为第一个经过的不合法点的路径。

第三个红点，也是终点,$(3, 4)$.

$f[3]$先 $ += foto(1, 1, 3, 4)$，即$f[3] += 10$.

进入第二层循环。

枚举$j = 1$, $f[3] -= f[1] * foto(2, 2, 3, 4)$，即$f[3] -= 6$；去掉了$6$条先经过了第一个红点，再经过这个点的路径。

此时$f[3] = 4$;

枚举$j = 2$, $f[3] -= f[2] * foto(2, 3, 3, 4)$, 即$f[3] -= 2$；去掉了$2$条先经过了第二个红点，再经过这个点的路径。

枚举完毕，此时$f[3] = 2$。将$2$输出即为答案。

总结一下思路：

  1. 将点按照$x$坐标的升序排序，枚举要有序您说是吧；
  2. 预处理逆元，用来算组合数。(lucas)由于本篇博文主要讲的是这题用到的独特的思路和这题的解法，不太注重算法的讲解，所以请大家自行搜索组合数的求法，敬请谅解；
  3. 接下来就是两层循环。
    首先要让$f[i]$加上$foto(1, 1, a[i].x, a[i].y)$，作为起点到该点的路径，接着循环枚举$j$,让$f[i] - f[j] * foto(a[i].x, a[i].y, a[j].x, a[j].y)$;
  4. 输出答案。
