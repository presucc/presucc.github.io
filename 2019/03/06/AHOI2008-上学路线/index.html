ТёЪУ░бУ┐Ўу»ЄжбўУДБ№╝їУ«ЕТѕЉС╗јСИГУјитЙЌС║єСИђС║ЏТќ░уџёТђЮУи»тњїТ│еТёЈтѕ░С║єтЁХС╗ќТќ╣жЮбуџёуѓ╣№╝їУђїУ┐ЎС║ЏтЈ»С╗ЦС╝ўтїќТЌХжЌ┤тцЇТЮѓт║дуџёуѓ╣Тў»тЅЇжЮбжбўУДБТ▓АТюЅУ»┤тѕ░уџёсђѓжѓБТѕЉт░▒УАЦтЁЁСИђСИІсђѓ

ТЅђС╗ЦТѕЉС╗гТюђтЦйућеСИі$dijkstra$сђѓ№╝ѕСйєТѕЉУ┐ўТў»ућеС║єspfa№╝їтЋфТЅЊУёИ№╝Ѕ

тЁХТгА№╝їт░▒Тў»тѕєС║ФСИђСИІТѕЉтдѓСйЋТЃ│тѕ░У┐ЎжбўуџёТђЮУи»уџёсђѓ

Та╣ТЇ«жбўТёЈ№╝їУ┐ЎжЂЊжбўТюЅСИцСИфжЌ«№╝џ

ТѕЉС╗гжђџУ┐Єућ╗тЏЙТЮЦТЅЙтЄ║Тќ╣Т│Ћсђѓ

У┐ЎТў»ТаиСЙІуџёжѓБт╣ЁтЏЙ№╝џ№╝ѕТ»ЈТЮАУи»тЙётидУЙ╣уџёТЋ░тГЌТў»УиЮуд╗№╝їтЈ│УЙ╣уџёТў»С╗БС╗и№╝Ѕ

СИЇжџЙтЙЌтЄ║№╝їуггСИђжЌ«уџёуГћТАѕт░▒Тў»2 №╝ѕ$1->2->6$ or $1->5->6$№╝Ѕ

уёХтљј№╝їуггС║їжЌ«уџёуГћТАѕт░▒Тў»Тіі$5->6$ тњї$2->6$СИцТЮАУЙ╣тѕатј╗№╝їУЙЊтЄ║$2+3=5$

т»╣С║југгСИђжЌ«№╝їТЎ«жђџТюђуЪГУи»тЙёТ▒ѓтЇ│тЈ»сђѓFloydтѕџтЦйтЈ»С╗ЦтЇАУ┐Єтј╗500№╝їСйєТў»УђЃУЎЉтѕ░СИІжЮбУ┐ўТюЅСИфТюђт░ЈтЅ▓УдЂтЂџ№╝їт░▒СИЇУдЂућеУ┐ЎуДЇу«ЌТ│Ћсђѓ

СйєТў»уггС║їжЌ«ТюЅуѓ╣ж║╗уЃдсђѓ

уюІтѕ░жбўжЮбСИГуџё
РђюуёХУђїтЈ»тЈ»т┐йуёХТюЅС║єСИђСИфжг╝уѓ╣тГљ№╝їС╗ќТЃ│УХЂтЇАтЇАСИЇтцЄ№╝їтюетЇАтЇАуџёУЙЊтЁЦТЋ░ТЇ«СИГтѕатј╗СИђС║ЏУи»у║┐№╝їС╗јУђїУ«ЕтЇАтЇАуџёуеІт║ЈтЙЌтЄ║уџёуГћТАѕтцДС║јт«ъжЎЁуџёТюђуЪГТЌХжЌ┤РђЮУ┐ЎтЈЦУ»ЮСИГуџёРђютѕаРђЮ№╝їжѓБС╣ѕСИђСИфтюетЏЙСИГуџёу«ЌТ│ЋСЙ┐Тх«уј░тюеТѕЉС╗гУёЉТхижЄї№╝џТюђт░ЈтЅ▓сђѓ

Тюђт░ЈтЅ▓т«џС╣Ѕ№╝џтюеСИђСИфуйЉу╗южЄїжЮб№╝їтдѓТъюУЃйтѕаТјЅТЪљС║ЏУЙ╣№╝їСй┐тЙЌуйЉу╗юСИЇУ┐ъжђџ№╝їжѓБС╣ѕУ┐ЎСИфУЙ╣жЏєт░▒тЈФРђютЅ▓РђЮсђѓтюеТ╗АУХ│РђюСй┐тЙЌуйЉу╗юСИЇУ┐ъжђџРђЮуџётЅЇТЈљСИІ№╝їтЙЌтѕ░СИђу╗ёС╗БС╗иТюђт░ЈуџёУЙ╣жЏє№╝їУ┐ЎСИфУЙ╣жЏєт░▒тЈФтЂџРђюТюђт░ЈтЅ▓РђЮсђѓ

СйєТў»№╝їТюђт░ЈтЅ▓С╗ЁС╗ЁТў»тј╗ТјЅТЪљС║ЏУЙ╣№╝їтњїжбўуЏ«ТёЈТђЮСИЇугдсђѓТђјС╣ѕтіътЉб№╝Ъ

ТѕЉС╗гУђЃУЎЉТііУ┐ЎСИфтЏЙуџёТюђуЪГУи»тЏЙТ▒ѓтЄ║ТЮЦ№╝їтюеТюђуЪГУи»тЏЙСИіТ▒ѓТюђт░ЈтЅ▓сђѓ

С╗ђС╣ѕТў»ТюђуЪГУи»тЏЙ№╝Ъ

Тў»ућ▒У«ИтцџТюђуЪГУи»ТъёТѕљуџётЏЙтЋісђѓсђѓсђѓС╣Ът░▒Тў»уюЂтј╗С║єСИђС║ЏТ▓АућеуџёУЙ╣№╝їТђјС╣ѕтѕцТќГтЉб№╝Ъ

тѕцТќГТќ╣Т│Ћ

тЁиСйЊТќ╣Т│Ћт░▒Тў»ждќтЁѕУиЉСИђТгАspfa

уёХтљјт░▒Т▒ѓС║єтЄ║ТЮЦтЇЋТ║љуџёТюђуЪГУи»$(sssp)$С║є№╝їуёХтљјТ▒ѓТюђуЪГУи»тЏЙ№╝џтдѓТъют»╣С║јСИђТЮАУЙ╣№╝їСИђСИфуФ»уѓ╣тѕ░1уџёУиЮуд╗+У┐ЎСИфУЙ╣уџёУЙ╣ТЮЃ==тЈдСИђСИфуФ»уѓ╣тѕ░$1$уџёУиЮуд╗№╝їт░▒У┐ъУЙ╣тѕ░уйЉу╗южЄїжЮбсђѓ№╝ѕтЈ»С╗ЦУЂћу│╗ТюђСИіжЮбжѓБСИфт╝ЈтГљ№╝Ѕ

Тюђтљј№╝їт░▒Тў»УдЂУ»ЂТўјТюђт░ЈтЅ▓==ТюђтцДТхЂС║єсђѓ№╝ѕтЈ»УЃйтЮљтюеућхУёЉтЅЇуџёуЦъС╗ЎТѓеуюІтѕ░У┐ЎтЈЦУ»ЮжЕгСИіт░▒тј╗тєЎТюђтцДТхЂС║є№╝Ѕ

УњЪУњ╗ТѕЉтйЊуёХСИЇС╝џУ»ЂТўј№╝їС║јТў»т░▒УйгУййС║єуйЉСИіСИђСйЇтєЎуџёжЮътИИтЦйуџёСИђСйЇтцДуЅЏуџёУ»ЂТўјУ┐ЄуеІ№╝џ

У»ЂТўјУ┐ЄуеІ№╝џ

тЁиСйЊуџёУ»ЂТўјтѕєСИЅжЃетѕє

1.С╗╗ТёЈСИђСИфТхЂжЃйт░ЈС║јуГЅС║јС╗╗ТёЈСИђСИфтЅ▓

У┐ЎСИфтЙѕтЦйуљєУДБ УЄфТЮЦТ░┤тЁгтЈИжџЈСЙ┐у╗ЎСйат«Хжђџуѓ╣Т░┤№╝їТъёТѕљСИђСИфТхЂ

ТЂљТђќтѕєтГљжџЈСЙ┐уаЇтЄатѕђ уаЇтЄ║СИђСИфтЅ▓

ућ▒С║јт«╣жЄЈжЎљтѕХ№╝їТ»ЈСИђТа╣уџёУбФуаЇуџёТ░┤у«АтГљТхЂтЄ║уџёТ░┤ТхЂжЄЈжЃйт░ЈС║ју«АтГљуџёт«╣жЄЈ

Т»ЈСИђТа╣УбФуаЇуџёТ░┤у«АуџёТ░┤ТюгТЮЦжЃйУдЂтѕ░Сйат«Хуџё№╝їуј░тюеТхЂтѕ░тцќжЮб тіаУхиТЮЦтЙЌтѕ░уџёТхЂжЄЈУ┐ўТў»уГЅ
С║јтјЪТЮЦуџёТхЂ

у«АтГљуџёт«╣жЄЈтіаУхиТЮЦт░▒Тў»тЅ▓№╝їТЅђС╗ЦТхЂт░ЈС║јуГЅС║јтЅ▓

ућ▒С║јСИіжЮбуџёТхЂтњїтЅ▓жЃйТў»С╗╗ТёЈТъёжђауџё№╝їТЅђС╗ЦС╗╗ТёЈСИђСИфТхЂт░ЈС║јС╗╗ТёЈСИђСИфтЅ▓

2.ТъёжђатЄ║СИђСИфТхЂуГЅС║јСИђСИфтЅ▓

тйЊУЙЙтѕ░ТюђтцДТхЂТЌХ№╝їТа╣ТЇ«тбът╣┐Уи»т«џуљє

Т«ІуЋЎуйЉу╗юСИГ$s$тѕ░$t$ти▓у╗ЈТ▓АТюЅжђџУи»С║є№╝їтљдтѕЎУ┐ўУЃйу╗Ду╗Гтбът╣┐

ТѕЉС╗гТііsУЃйтѕ░уџёуџёуѓ╣жЏєУ«ЙСИ║$S$,СИЇУЃйтѕ░уџёуѓ╣жЏєСИ║$T$

ТъёжђатЄ║СИђСИфтЅ▓жЏє$C[S,T]$,$S$тѕ░$T$уџёУЙ╣т┐ЁуёХТ╗АТхЂ тљдтѕЎт░▒УЃйу╗Ду╗Гтбът╣┐

У┐ЎС║ЏТ╗АТхЂУЙ╣уџёТхЂжЄЈтњїт░▒Тў»тйЊтЅЇуџёТхЂтЇ│ТюђтцДТхЂ

ТііУ┐ЎС║ЏТ╗АТхЂУЙ╣СйюСИ║тЅ▓№╝їт░▒ТъёжђатЄ║С║єСИђСИфтњїТюђтцДТхЂуЏИуГЅуџётЅ▓

3.ТюђтцДТхЂуГЅС║јТюђт░ЈтЅ▓

У«ЙуЏИуГЅуџёТхЂтњїтЅ▓тѕєтѕФСИ║$Fm$тњї$Cm$

тѕЎтЏаСИ║С╗╗ТёЈСИђСИфТхЂт░ЈС║јуГЅС║јС╗╗ТёЈСИђСИфтЅ▓

С╗╗ТёЈ$FРЅцFm=CmРЅц$С╗╗ТёЈ$C$

т«џуљєУ»┤Тўјт«їТѕљ№╝їУ»ЂТўјтдѓСИІ№╝џ

т»╣С║јСИђСИфуйЉу╗юТхЂтЏЙ$G=(V,E)$№╝їтЁХСИГТюЅТ║љуѓ╣$s$тњїТ▒Єуѓ╣$t$№╝їжѓБС╣ѕСИІжЮбСИЅСИфТЮАС╗ХТў»уГЅС╗иуџё№╝џ

  1. ТхЂfТў»тЏЙGуџёТюђтцДТхЂ

  2. Т«ІуЋЎуйЉу╗юGfСИЇтГўтюетбът╣┐Уи»

  3. т»╣С║јGуџёТЪљСИђСИфтЅ▓(S,T)№╝їТГцТЌХf=C(S,T)

ждќтЁѕУ»ЂТўј$1 => $2№╝џ

ТѕЉС╗гтѕЕућетЈЇУ»ЂТ│Ћ№╝їтЂЄУ«ЙТхЂ$f$Тў»тЏЙ$G$уџёТюђтцДТхЂ№╝їСйєТў»Т«ІуЋЎуйЉу╗юСИГУ┐ўтГўтюеТюЅтбът╣┐Уи»$p$№╝їтЁХТхЂжЄЈСИ║$fp$сђѓтѕЎТѕЉС╗гТюЅТхЂ$f$Рђ▓$=f+fp>f$сђѓУ┐ЎСИј$f$Тў»ТюђтцДТхЂС║ДућЪуЪЏуЏЙсђѓ

ТјЦуЮђУ»ЂТўј$2 => $3№╝џ

тЂЄУ«ЙТ«ІуЋЎуйЉу╗ю$Gf$СИЇтГўтюетбът╣┐Уи»№╝їТЅђС╗ЦтюеТ«ІуЋЎуйЉу╗ю$Gf$СИГСИЇтГўтюеУи»тЙёС╗ј$s$тѕ░УЙЙ$t$сђѓТѕЉС╗гт«џС╣Ѕ$S$жЏєтљѕСИ║№╝џтйЊтЅЇТ«ІуЋЎуйЉу╗юСИГ$s$УЃйтцЪтѕ░УЙЙуџёуѓ╣сђѓтљїТЌХт«џС╣Ѕ$T=V-S$сђѓТГцТЌХ$(S,T)$ТъёТѕљСИђСИфтЅ▓$(S,T)$сђѓСИћт»╣С║јС╗╗ТёЈуџё$uРѕѕS$,$vРѕѕT$№╝їТюЅ$f(u,v)=c(u,v)$сђѓУІЦ$f(u,v)<c(u,v)$№╝їтѕЎТюЅ$Gf(u,v)>0$№╝ї$s$тЈ»С╗Цтѕ░УЙЙ$v$№╝їСИј$v$т▒ъС║ј$T$уЪЏуЏЙсђѓ

тЏаТГцТюЅ$f(S,T)=╬Бf(u,v)=╬Бc(u,v)=C(S,T)$сђѓ

ТюђтљјУ»ЂТўј$3 => 1$№╝џ

ућ▒С║ј$f$уџёСИіуЋїСИ║Тюђт░ЈтЅ▓№╝їтйЊ$f$тѕ░УЙЙтЅ▓уџёт«╣жЄЈТЌХ№╝їТўЙуёХт░▒ти▓у╗Јтѕ░УЙЙТюђтцДтђ╝№╝їтЏаТГцfСИ║ТюђтцДТхЂсђѓ
